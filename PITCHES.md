## 4. Press Kit Pitch

### One-Liner
Last Rally is a modern Pong with a 13-quest campaign, four AI difficulties, and a full unlock systemâ€”playable instantly in any browser.

### The Hook
We asked: what if Pong had been designed with modern progression systems? Not loot boxes or battle passesâ€”just honest skill gates. Beat Easy AI 5 times to unlock the green paddle. Complete all 13 quests for the rainbow trail. Simple rewards for real accomplishments.

### Why It Matters
In an era of 100-hour games and attention-hungry live services, Last Rally is a counterpoint. A complete experience in 2-minute sessions. Depth without bloat. The entire game fits in a browser tab and respects your time.

### Key Features
- **Instant Play**: One click to start. No tutorial unless you want it.
- **Real Progression**: 4 difficulties, 13 quests, 22 achievements, 14 cosmetics
- **Local Multiplayer**: Split keyboard (W/S vs I/K)
- **Zero Friction**: No accounts, no ads, saves to localStorage
- **Synthesized Audio**: Retro sounds generated in real-time

### Tech Notes
- Built with React + TypeScript + Canvas 2D
- 60fps game loop with particle effects
- Web Audio API for synthesized retro sounds
- ~75KB gzipped total bundle
